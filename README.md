#Projeto Backend com Python 

## Objetivos do Projeto  
- Criar as **primeiras rotas utilizando FastAPI**
- Entender o fluxo de funcionamento de uma **API backend**
- Trabalhar com **regras de negócio**

- Evoluir gradualmente a arquitetura do código


## Tecnologias
